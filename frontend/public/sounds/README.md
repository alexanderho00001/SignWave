# Sound Effects

This directory contains sound effects for the app.

## success.mp3
Success/ding sound that plays when user correctly signs a letter/number/word.

You can replace this with any MP3 file. Free sound effects can be found at:
- https://pixabay.com/sound-effects/
- https://freesound.org/
- https://mixkit.co/free-sound-effects/

For now, the app uses a web audio API generated tone if the file is not found.
